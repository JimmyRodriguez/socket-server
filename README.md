# Socket-Server 

1. Reconstruir modulos de Nodes

   - npm install socket

2. Generar el Dist

   - tsc -w

3. Levantar Servidor - cualquiera de las dos opciones
    - nodemon dist/
    - node dist/




4. Observaciones de Instalaciones.
    1. Tener instalado TypeScript
    2. Tener instalado nodemon
